trivialUtils
============

Some day to day utilities based on some basic shell and python scripting

Utilites
--------
1. booklet printer :

    A shell script utility to print a PDF document in a booklet.

    #Usage:  

        $trivialUtilsRepo/bin/print_booklet.sh FILE

    #Dependencies :
     1. psutils
     2. pdfjam
     3. pdftk
